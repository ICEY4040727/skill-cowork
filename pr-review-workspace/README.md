# pr-review-workspace

This workspace is reserved for runtime evaluation outputs produced while exercising the published `pr-review` skill.

## What belongs here

- Iteration outputs generated from `pr-review/evals/evals.json`
- Run artifacts such as `timing.json`, `grading.json`, and benchmark outputs

## What does not belong here

- Skill-authoring methodology experiments (for example, A/B tests about whether examples should be included in SKILL.md)

Those authoring experiments are stored under:
- `research/pr-review-example-ablation/`
